Home Page

The Home Page acts as the central entry point of the website, introducing the theme "Life on Land" and providing users with an overview of key environmental topics. The design focuses on clarity, accessibility, and visual appeal to immediately engage users.

A hero section with a large background title ("Land Verse") and a short tagline creates a strong first impression. This is followed by a structured "Mission & Goals" section that presents four main topics: forest protection, soil health restoration, wildlife conservation, and sustainable land use.

Each topic is displayed using a card layout. These cards include images, headings, descriptions, and actionable bullet points. This approach improves readability and allows users to quickly understand key ideas without overwhelming them with text.

One of the cards is interactive and links directly to a content page, improving navigation and encouraging users to explore deeper information. The layout was designed using Flexbox to ensure proper alignment and responsiveness across different screen sizes.

The navigation bar at the top provides easy access to all pages, while the footer includes contact information and additional links, ensuring a complete and user-friendly structure.

Gallery Page

The Gallery Page is designed to visually represent important environmental topics through images and short descriptions. It enhances user engagement by combining visual elements with informative content.

The page displays multiple image cards, each containing a title, image, and descriptive text. These topics include forest conservation, wildlife protection, reforestation, and land degradation.

A key feature of this page is the interactive modal system implemented using JavaScript. When a user clicks on an image, a popup (modal) appears showing a larger version of the image along with its title and description.

This functionality improves user experience by allowing users to focus on specific content without navigating away from the page. It also makes the website more dynamic and interactive.

The layout is structured using CSS for spacing and alignment, ensuring that images are displayed neatly and consistently across different screen sizes.

Technical Description

The website was developed using HTML for structure, CSS for styling, and JavaScript for interactivity. External CSS files (global.css, home.css, gallery.css) were used to maintain consistency and improve code organization.

Flexbox was used for layout design, particularly for aligning navigation elements and arranging content sections. This allowed for a responsive design that adapts to different screen sizes.

The Gallery Page uses JavaScript to handle user interactions. Event listeners detect when an image is clicked, and dynamically update the modal content (image source, title, and description).

The navigation system is consistent across all pages, improving usability. Reusable components like headers and footers were implemented to maintain a uniform design throughout the website.

Challenges and Lessons Learned

One of the main challenges was ensuring consistency across multiple pages while using different CSS files. Maintaining the same design style required careful planning and reuse of common classes.

Another challenge was implementing the modal functionality for the gallery. It required understanding how JavaScript interacts with HTML elements and how to dynamically update content.

I also faced difficulties with spacing, alignment, and responsiveness. These were solved by using Flexbox and testing the layout on different screen sizes.

Through this project, I improved my skills in front-end development, especially in structuring web pages, managing styles, and creating interactive features.
