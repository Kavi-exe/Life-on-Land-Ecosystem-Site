Splash Screen

The Splash Screen is the first interaction point of the website. It introduces the "Life on Land" theme using a visually engaging layout. The structure is built using basic HTML elements such as <div>, <h1>, and <p>.

CSS animations are used to create smooth transitions and visual effects. Properties such as @keyframes, opacity, and transform enhance the appearance.

JavaScript is used to automatically redirect users to the Home Page using setTimeout(), ensuring a seamless experience.

The Splash Screen does not use the shared template but maintains consistency with the rest of the website by using the same color scheme, fonts, and design elements. This ensures a uniform look and feel across all pages.

Overall, the Splash Screen combines structured HTML, modern CSS styling, and JavaScript functionality to create an engaging introduction that enhances the overall user experience of the website.

SDG Action Impact Simulator

The SDG Action Impact Simulator allows users to interact with environmental actions and see their impact score. The layout is created using <section>, <div>, and <button>.

JavaScript event listeners detect user input and dynamically update the score using DOM manipulation.

CSS Flexbox is used for layout alignment, while hover effects improve interactivity.

The simulator helps users understand the importance of sustainable actions by demonstrating how small changes can lead to a significant positive impact on the environment. It also increases user engagement by making the learning process interactive and practical.

Overall, the SDG Action Impact Simulator combines structured HTML, styled components using CSS, and dynamic functionality through JavaScript to create an effective and engaging educational tool.

Content Page – Sustainable Land use

This page focuses on the concept of sustainable land use and its importance in maintaining a balance between human development and environmental protection. It is structured using semantic HTML elements such as <section> and <h2> to organize content clearly and improve accessibility.

A split layout is implemented using text and image containers, which are aligned side by side using CSS Flexbox. This layout improves readability and creates a visually appealing structure for users.

To avoid a repetitive design, alternating layouts are applied using flex-direction: row-reverse;, allowing images and text to switch positions across sections. This enhances the overall user experience and visual flow of the page.

The content highlights key aspects of sustainable land use, including the importance of balancing development with nature, protecting biodiversity, and minimizing environmental damage caused by human activities. It also discusses major threats such as deforestation, urbanization, and unsustainable farming practices.

In addition, the page explains how sustainable land use contributes to climate regulation by reducing carbon emissions and promoting practices like reforestation and soil conservation. It emphasizes the role of responsible land management in preventing environmental issues such as soil degradation, floods, and droughts.

Overall, this page combines structured content, responsive design, and interactive features to effectively communicate the importance of sustainable land use and encourage users to adopt responsible and environmentally friendly practices.

Technical Description

The development of the Splash Screen, SDG Action Impact Simulator, and the "Restore Soil Health" content page involved the use of HTML, CSS, and JavaScript to create a structured, styled, and interactive web experience.

HTML was used to define the structure of each page using semantic elements such as <header>, <main>, <section>, and <footer>. These elements improve accessibility and ensure a clear organization of content.

CSS was applied to control the visual presentation of the pages. Layout techniques such as Flexbox (display: flex;) were used to align content, manage spacing, and create responsive designs. Additional styling properties such as margin, padding, border-radius, and box-shadow were used to enhance the overall appearance.

The Splash Screen uses CSS animations defined with @keyframes to create smooth transitions such as fade-in and loading effects. JavaScript is used with the setTimeout() function to automatically redirect users to the Home Page after a few seconds.

The SDG Action Impact Simulator is implemented using JavaScript to handle user interactions. Event listeners are attached to buttons, and functions are used to update the impact score dynamically. DOM manipulation techniques such as document.getElementById() and innerText are used to display updated values without reloading the page.

The "Restore Soil Health" content page uses a combination of structured HTML and CSS layout techniques to present information clearly. Flexbox is used to create side-by-side layouts, while classes such as reverse are used to alternate content positioning for improved visual design.

Overall, the integration of HTML, CSS, and JavaScript ensures that the website is well-structured, visually appealing, responsive, and interactive.

Challenges and Lessons Learned

One of the main challenges faced during development was implementing the JavaScript functionality for the SDG Action Impact Simulator. Managing dynamic score updates required a clear understanding of event handling and DOM manipulation.

Another challenge was creating smooth animations for the Splash Screen. Ensuring that animations were visually appealing without affecting performance required careful use of CSS properties such as opacity and transform.

Designing a responsive layout for the "Restore Soil Health" content page was also challenging. Aligning text and images properly across different screen sizes required the effective use of Flexbox and spacing techniques.

Maintaining consistency across multiple pages was another difficulty. It required careful reuse of styles and design elements to ensure a uniform appearance throughout the website.

Through this project, I improved my skills in front-end development, particularly in structuring web pages using semantic HTML, styling layouts with CSS, and adding interactivity using JavaScript.

Overall, these challenges helped me gain practical experience and a deeper understanding of modern web development techniques.

Compliance

This website was developed in accordance with the coursework requirements and guidelines. All pages follow a consistent layout structure, including a navigation bar, main content area, and footer, ensuring a uniform user experience.

The design prioritizes accessibility by using clear fonts, readable text sizes, and appropriate color contrast between text and background. Navigation is simple and intuitive, allowing users to easily move between pages.

The website is responsive and adapts to different screen sizes using modern CSS techniques such as Flexbox. This ensures that content is displayed properly on desktops, tablets, and mobile devices.

All images used in the project are sourced from copyright-free platforms, and proper references have been included. No copyrighted material has been used without permission.

External CSS and JavaScript files are properly linked, and the code is organized to improve readability and maintainability.

Overall, the website meets web development standards and follows best practices in design, usability, accessibility, and content organization.
